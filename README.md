# Virat Kohli Statistics Dashboard (Power BI)

This project presents an interactive Power BI dashboard analyzing the career statistics of Virat Kohli.

## Tools Used
- Power BI
- Microsoft Excel

## Features
- Runs analysis
- Match-wise performance
- Year-wise comparison
- Interactive filters and visuals

## Dataset
The dataset includes batting statistics collected from reliable sources.
